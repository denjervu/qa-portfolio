# Задача
В веб-приложении есть выпадающий список с выбором стран и кодов телефона. Необходимо проверить корректность работы списка. Ожидаемый результат: Описание позитивных и негативных тест-кейсов.

# Примечание

В решении задачи ориентировался на функциональное тестирование для desktop и стремился к единообразию в названии кейсов.

Разбил проверки на категории по функционалу.

Не тестировал:
- корректность отображения
- корректность значений списка
- адаптивность

Тестировал выпадающий список без поиска и ввода страны с клавиатуры.
Пример выпадающего списка: https://www.airbnb.co.in/signup_login

В конце привел пояснения к названиям элементов выпадающего списка.

# Описание тест-кейсов
## Раскрытие Dropdown Menu
1. Раскрытие Dropdown Menu происходит по клику на Select Box.
2. Раскрытие Dropdown Menu возможно с помощью клавиатуры (Tab и Enter).
3. Раскрытие Dropdown Menu происходит вниз, если расстояние от Select Box до нижней границы области просмотра больше, чем до верхней.
4. Раскрытие Dropdown Menu происходит вверх, если расстояние от Select Box до верхней границы области просмотра больше, чем до нижней.
5. Раскрытие Dropdown Menu не происходит при клике на отключенный Select Box.
## Выбор элемента
1. Выбор активного элемента Dropdown Menu добавляет название элемента в Select Box.
2. Выбор неактивного элемента Dropdown Menu невозможен.
3. Повторный выбор нового элемента в Dropdown Menu обновляет название элемента в Select Box.
4. Повторный выбор уже выбранного элемента в Dropdown Menu сохраняет название элемента в Select Box.
5. Перезагрузка страницы не сбрасывает название элемента в Select Box.

## Навигация по списку
1. Пролистывание Scrollable area возможно с помощью колесика мыши.
2. Пролистывание Scrollable area возможно с помощью наведения курсора на края Scrollable area, что вызывает пролистывание элементов.
3. Пролистывание Scrollable area возможно с помощью стрелок на клавиатуре.
4. Пролистывание Scrollable area возможно с помощью стрелок (Scroll up/down arrow) на Scrollbar.
5. Пролистывание Scrollable area возможно с помощью ползунка прокрутки (scroll thumb).
6. Пролистывание Scrollable area возможно с помощью клика на Scroll track.

## Закрытие списка
1. Закрытие Dropdown Menu происходит при выборе элемента.
2. Закрытие Dropdown Menu происходит при клике на область вне Dropdown Menu.
3. Закрытие Dropdown Menu происходит, когда Select Box выходит за пределы экрана при пролистывании страницы.
4. Пролистывание Scrollable area не вызывает её закрытие.

# Обозначение элементов выпадающего списка

| Dropdown              | Выпадающий список                                                                                  |
| --------------------- | -------------------------------------------------------------------------------------------------- |
| Select Box            | Область, которая открывает Dropdown Menu при клике. Select Box содержит текущее выбранное значение |
| Dropdown Menu         | Область, которая открывается по клику на Select Box. Состоит из: Scrollable area + Scrollbar       |
| Scrollable area       | Область контента, которая состоит из элементов.                                                    |
| Scrollbar             | Scroll thumb + scroll track + scroll arrows                                                        |
| Scroll **thumb**      | Ползунок прокрутки                                                                                 |
| Scroll **track**      | Полоса, по которой движется ползунок.                                                              |
| Scroll **up arrow**   | Кнопка со стрелкой вверх.                                                                          |
| Scroll **down arrow** | Кнопка со стрелкой вниз.                                                                           |
